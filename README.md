# Beyond Last-Click

Attribution analysis of influencer vs paid vs organic channels in e-commerce.

## Project
Analysing which marketing channel — influencer, paid social, or organic — 
drives the highest engagement and conversion value across YouTube, Instagram and TikTok.

## Data Sources
- YouTube Data API v3 — 1,805 real channels across all influencer tiers
- Marketing A/B Testing dataset — 588,101 rows of real experimental data

## Tech Stack
Python · SQL · Tableau · Jupyter · GitHub
